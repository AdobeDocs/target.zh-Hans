# 用于控制多个特征的特征组 {#feature-groups}

[功能标志](what-is-a-feature-flag.md)控制单个功能。 当您需要一起管理多个相关功能标记并确保它们访问同一受众时，您使用&#x200B;**功能组**。

## 功能组的功能 {#what-it-does}

功能组捆绑了多个功能标志，并允许您一次性将单个受众规则应用于所有受众。 当单个面向用户的功能跨越多个应用程序或服务时，此功能非常有用。

例如，考虑一个涉及跨桌面应用程序、移动应用程序和后端服务进行更改的协作功能。 通过创建一个包含所有三个应用程序中的标记的功能组，您可以向1%的用户打开该功能，并确保这些用户同时在所有三个表面看到一致的体验。

## 跨应用程序分组 {#cross-application}

功能组支持跨应用程序功能管理。 可以将跨多个应用程序的相关标志组合在一起。
