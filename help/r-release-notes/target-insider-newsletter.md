# Target Insider 新闻稿

Adobe Target Insider 是针对 Adobe Target 社区成员发布的每月时事通讯。可让您了解产品更新和未来计划、个性化和优化提示与技巧、客户成功案例、即将举行的活动、信息丰富的白皮书及热门博客文章，等等。


请阅读[公告函](https://theblog.adobe.com/stay-optimized-adobe-target-insider-newsletter/)以了解如何使用 Adobe Target Insider 时事通讯保持优化。
