# 管理应用程序 {#manage-applications}

Flags中的&#x200B;**应用程序**&#x200B;表示您要使用功能标志控制的服务或产品。 在创建功能标记之前，必须将至少一个应用程序载入控制台。

## 谁可以管理应用程序 {#permissions}

只有具有&#x200B;**管理员**&#x200B;角色的成员才能添加或修改应用程序。 如果您没有此角色，请联系您的管理员。

## 添加应用程序 {#add-application}

要添加应用程序，请参阅[载入应用程序](onboard-your-application.md)以了解分步说明。
