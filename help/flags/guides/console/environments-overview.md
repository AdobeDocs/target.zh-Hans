# 选择您的沙盒 {#sandbox}

标记构建于Adobe Experience Platform之上。 在使用功能标记之前，请选择与当前环境对应的沙盒，就像在任何其他Adobe Experience Platform应用程序中一样。

## 如何选择沙盒 {#how-to}

使用标志控制台顶部导航栏中的沙盒切换器，在创建或修改功能标志之前选择正确的沙盒。
