# [!DNL Experience Cloud Device Co-op]{#experience-cloud-device-co-op}

此信息介绍了 [!DNL Adobe Experience Cloud Device Co-op]，该计划允许参与者通过合作更好地识别跨多个数字接触点的用户，同时确保最高级别的隐私和透明度。

[!DNL Experience Cloud Device Co-op] 让参与品牌能够识别他们的客户，以便他们在许多设备和应用程序上提供更加个性化的体验。[!DNL Device Co-op] 是 [!DNL Adobe Experience Cloud] 的一项核心服务。使用 [!DNL Analytics]、[!DNL Audience Manager]、[!DNL Media Optimizer] 或 [!DNL Target] 的 [!DNL Adobe] 客户可以使用此功能。有关更多信息，请参阅 [Adobe Experience Cloud 设备协作文档](https://marketing.adobe.com/resources/help/zh_CN/mcdc/)。

在 [!DNL Target] 中，您可以在 A/B 测试、多变量 (MVT) 测试和体验定位活动中使用 [!DNL Device Co-op] 数据。在[!UICONTROL 目标和设置]页面上通过 [!DNL Target] 三步引导式工作流创建活动时，可以使用[!UICONTROL 设备协作]选项。有关更多信息，请参阅 [Target：A/B 测试、多变量测试和体验定位](https://marketing.adobe.com/resources/help/zh_CN/mcdc/mcdc-target.html)。
