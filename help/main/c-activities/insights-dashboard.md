# Adobe Target分析功能板

[!UICONTROL Adobe Target仪表板]提供了贵组织如何逐渐使用[!DNL Adobe Target]的高级视图。 它有助于团队一眼就了解采用情况、活动量和实验使用情况。

该仪表板专为希望快速了解[!DNL Target]使用情况的从业人员和利益相关者设计，无需深入了解各个活动报告。

查看此仪表板时，请牢记以下事项：

* 量度可能包括在所选时间范围之前开始或之后结束的活动。
* 根据活动的生命周期（例如，已发布和已完成），可在多个量度中统计该活动。
* 该仪表板侧重于使用和采用，而不是性能结果。

有关详细的结果、提升度或统计性能，请参阅[!DNL Adobe Target]中的[各个活动报告](../c-reports/reports.md)。

![](assets/insights-1.png)

## [!UICONTROL Experimentation Accelerator]

仪表板上的横幅提供对&#x200B;**[!UICONTROL Experimentation Accelerator]**&#x200B;的直接访问，它是一个轻量级的入口点，可用于简化试验工作流并简化试验设置、分析和决策。

## 时间范围选择

要设定仪表板上显示的数据的范围，请选择一个时间范围，例如上周、去年或所有时间。 选定的时间范围始终适用于功能板上的所有量度和图表。

![](assets/insights-2.png)

在解释选定时间范围内的量度时，请牢记以下几点：

* 某些量度反映了该时间范围内任何时间的实时活动。

* 其他则反映在该时间范围内创建、发布或完成的活动。

* 因此，各指标的总数可能无法准确累加。 例如，可以在同一时间范围内开始和完成许多活动。

您还可以通过从高级菜单中选择&#x200B;**[!UICONTROL 以PNG格式下载]**&#x200B;导出仪表板的快照。

![](assets/insights-3.png)

## 量度

仪表板将其量度组织为四个互补视图，每个视图回答有关您[!DNL Target]使用情况的不同问题：[KPI](#kpis)提供活动计数概览，[活动类型细分](#activity-type-breakdown)显示您最依赖的功能，[A/B测试量度](#ab-testing-metrics)放大实验使用情况，以及[随时间变化的活动](#activities-over-time)显示选定时间范围内的趋势。

### KPI

页面顶部的KPI卡片概述了所选时间范围内的关键活动计数。 每张信息卡都重点关注活动生命周期的不同阶段（实时阶段、修改阶段、结束阶段或发布阶段），因此您可以快速评估整体使用情况和动向。

![](assets/insights-4.png)

**实时活动总数**&#x200B;指标详细列出了选定时间范围内任何时间点上实时的活动数。 如果活动积极提供流量，则即使该活动在选定时段之前开始或之后结束，该活动也被视为实时活动。 使用此量度可以：

* 了解在此时间段内[!DNL Target]的使用情况。
* 衡量个性化和测试工作的整体规模。

**实时或修改的活动**&#x200B;量度表示贵组织在所选时间范围内实时、创建或修改的活动总数。 使用此量度可以：

* 了解[!DNL Target]活动库的总大小以及正在使用的活动数量。

* 跟踪您的试验和个性化项目的长期增长。

**结束的活动**&#x200B;量度表示在所选时间范围内达到完成或停止日期的活动数。 使用此量度可以：

* 了解在该时间段内结束的活动量。
* 跟踪一段时间的完成量。

**已发布活动**&#x200B;度量详细列出了选定时间范围内已发布的活动数。 首次将活动上线时，该活动会被视为已发布。 如果某个活动处于实时状态、停止状态，然后又重新处于实时状态，则此量度中只会计入第一次发生的事件。 使用此量度可以：

* 测量已启动的新活动数。
* 了解活动创建和发布速度。

### 活动类型细分

![](assets/insights-5.png)

[!UICONTROL 活动类型]图表显示所选时间范围内按类型列出的实时活动分布，包括：

* [!UICONTROL A/B 测试]
* [!UICONTROL 体验定位]
* [!UICONTROL 推荐]
* [!UICONTROL Automated Personalization]
* [!UICONTROL 多变量测试]

使用此图表可以识别贵组织最依赖的[!DNL Target]功能，并发现扩大您运行的活动类型组合的机会。

### A/B测试指标

![](assets/insights-6.png){align="center"}

本节重点介绍与&#x200B;**[!UICONTROL A/B测试]**&#x200B;活动具体相关的用法。

**[!UICONTROL 实时A/B测试活动总数]**&#x200B;指标显示在所选时间范围内的任意时间点上线的&#x200B;**[!UICONTROL A/B测试]**&#x200B;活动数。

发布的&#x200B;**[!UICONTROL A/B测试总数]**&#x200B;显示在所选时间范围内发布的&#x200B;**[!UICONTROL A/B测试]**&#x200B;活动数。

使用这些量度了解A/B测试的使用频率，并跟踪一段时间内的试验量和采用情况。

### 一段时间内的活动

![](assets/insights-7.png){align="center"}

**[!UICONTROL 随时间变化的活动]**&#x200B;图形可跟踪在选定时间范围内创建、修改和发布的活动数，从而轻松地在您的试验项目中发现趋势、峰值或安静时段。
