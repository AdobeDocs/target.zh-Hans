# 导入功能标志 {#import-feature-flags}

利用标志，可将功能标志从一个沙盒（例如，沙盒1）导入另一个沙盒（例如，沙盒2）中。 这可避免手动重新创建标记配置，并减少沙盒之间的配置漂移风险。

## 步骤1：转到目标沙盒和应用程序 {#step-1}

登录到&#x200B;**目标**&#x200B;沙盒的控制台 — 您要将标志&#x200B;*导入到*&#x200B;中的沙盒。 从“功能标志”页上的应用程序下拉列表中选择要将标志导入到的应用程序。

>[!IMPORTANT]
>
>您当前的沙盒和选定的应用程序必须是&#x200B;**目标** — 而不是源。 例如，要将标志从sandbox 1导入sandbox 2，请登录sandbox 2控制台并选择您的sandbox 2应用程序。

## 第2步：打开导入对话框 {#step-2}

选择&#x200B;**导入功能标志**。 此时将打开一个对话框，其中显示了基于可用应用程序预填充的源沙盒和应用程序。 如果需要，您可以从对话框中的下拉列表更改源沙盒和应用程序。

## 第3步：选择要导入的功能标志 {#step-3}

从源沙盒中的功能标记列表中，选择要导入的标记。 您可以一次选择一个、多个或所有标志。

## 步骤4：选择要导入的功能标志的状态 {#step-4}

使用下拉菜单选择应如何导入功能标志 — **已启用**、**已禁用**&#x200B;或处于&#x200B;**当前状态**。 默认情况下，功能标志以&#x200B;**已禁用**&#x200B;状态导入。

## 重要说明 {#important-notes}

在导入功能标记时，请牢记以下几点：

* 如果目标沙盒中已存在具有相同键的功能标记，则不会导入该功能标记。
