# 操作系统

您可以通过以下方式定位访客： [!DNL Adobe Target] 使用特定操作系统的用户。

1. 在 [!DNL Target] 界面中，单击&#x200B;**[!UICONTROL 受众]** > **[!UICONTROL 创建受众]**。
1. 为受众命名并添加可选描述。
1. 拖放 **[!UICONTROL 操作系统]** 放入受众生成器窗格。
1. 单击&#x200B;**[!UICONTROL 选择]**，然后选择以下选项之一：

   * Linux
   * Macintosh
   * Windows

1. （可选）为受众设置其他规则。
1. 单击&#x200B;**[!UICONTROL 完成]**。
